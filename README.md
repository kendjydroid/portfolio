# Personal Portfolio - JEAN LOUIS Stéphane Richeca 

A responsive personal portfolio website built from scratch using vanilla HTML, CSS, and JavaScript. This is the first version of my online presence as Computer Science student pursuing two complementary specialization - Full-Stack Development | Cybersecurity.

## Live Demo 

[View live portfolio](https://kendjydroid.github.io/portfolio/)

## About 

I am a Computer Science student pursuing two complementary specialization tracks :
- **Full-Stack Software Development** - through the IBM Full-Stack Developer Professional Certificate on Coursera 
- **Cybersecurity** - through TryHackMe, with planned progression to HackTheBox 

This portfolio serves as both a showcase of my work and a milestone in my journey toward becoming a software engineer.


## Tech Stack 

- **HTML5** - Semantic markup with accessible structure 
- **CSS3** - Custom design tokens, Flexbox, Grid, responsive media queries
- **JavaScript** - DOM manipulation, event handling, form validation 
- **Git & GitHub** - Version control with Conventional commits 
- **Figma** - Wireframing and design 

## Features 

- Fully responsive design (desktop, tablet, mobile)
- Smooth scroll navigation between sections 
- Animated hamburger menu for mobile devices
- Contact form with client-side validation 
- Hover effects and subtitles micro-interactions 
- Modular CSS architecture using design token (CSS custom properties)
- Semantic HTML for accessibility 

## Sections
- **Hero** - Introduction and primary call-to-action
- **About** - Background and current certifications in progress 
- **Skills** - Technical competencies grouped by specialization 
- **Projects** - Showcase of completed and upcoming work 
- **Contact** - Form and social links 

## Running Locally
To view this project on your local machine :

```bash
# Clone the repository
git clone https://github.com/kendjydroid/portfolio.git

# Navigate to the project directory
cd portfolio

# Open index.html in your browser
# or use a local development server like VS Code Live Server
```
